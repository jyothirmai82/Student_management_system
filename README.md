# Student Management System

This is a Python-based application to manage student records using file handling.

## Features
- Add student
- View all students
- Search student by ID
- Delete student

## Technologies Used
- Python
- File Handling

## How to Run
1. Make sure Python is installed
2. Run the file:
   python student.py
3. Follow the menu options in terminal

## Sample Data Format
Each record is stored as:
ID, Name, Marks
Example:
1, Jyothi, 90
